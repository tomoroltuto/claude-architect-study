# Phase 5: 模擬問題・総合演習

## 学習範囲
- 公式 Exam Guide のサンプル問題 12問
- 6つの試験シナリオの理解と実践
- アンチパターンの総復習
- プロンプト評価（Prompt Evaluations）の実装

## 6つの試験シナリオ
1. Customer Support Resolution Agent
2. Code Generation with Claude Code
3. Multi-Agent Research System
4. Developer Productivity with Claude
5. Claude Code for CI/CD
6. Structured Data Extraction

> 試験では上記6つのうち4つがランダムに出題される。全6シナリオを学習すること。

## 重要なアンチパターン（試験頻出）
- [ ] stop_reason をテキスト解析で判定する（→ API の stop_reason を使う）
- [ ] サブエージェントがコーディネーターのコンテキストを継承する前提で設計する
- [ ] System Prompt だけで安全性を担保しようとする（→ コードで保証する）
- [ ] 任意のイテレーション上限を主要な停止メカニズムにする
