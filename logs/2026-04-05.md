### 学んだこと

#### Claudeデスクトップアプリの3つのモード

| 項目 | Chat | Cowork | Code |
|------|------|--------|------|
| 用途 | 素早い対話・アイデア探求・反復的な草稿作成 | 複雑・継続的な作業（調査・分析・文書作成） | ソフトウェアの構築（記述・テスト・デプロイ） |
| 主な機能 | クイック入力、音声入力 | ローカルフォルダ、プラグイン、サブエージェント、スケジュールタスク | 質問/コード/計画モード、ビジュアル差分、Git統合 |
| ツール | コネクタ、スキル、Claude for Chrome | コネクタ、スキル、Chrome、プラグイン | コネクタ、スキル、Chrome、プラグイン、フック |

- CoworkとCodeは内部的に同じエンジン（Claude Code）で動作している
- Coworkのスケジュールタスク機能は、毎朝の定型業務（メール確認・状況報告・会議準備など）の自動化に活用できそう

#### プロジェクト
- Claudeが参照すべき知識の保存、関連チャットの整理、チーム共同作業に使う機能
- 大量のファイルはRAGモードで最大10倍まで拡張可能
- 3段階の権限レベル: 閲覧可能 / 編集権限 / 所有者
- プロジェクトの共有も可能

**ベストプラクティス（5つ）:**
1. スモールベースで進める（幅広く取り込みすぎない）
2. 知識ベースを常に最新の状態に保つ
3. 明確な指示を書く（一貫性のない結果を防ぐ）
4. 関連する文書をグループ化する
5. 参照する文書の名称を必ず指定する（例:「弊社の第3四半期報告書に基づくと…」）

#### アーティファクト
- 会話の横に専用ウィンドウとして作成されるもの
- 種類: ドキュメント / コード / HTMLページ / SVG画像 / Mermaid図 / Reactコンポーネント
- 自動作成されない場合は「これを成果物として作成してください」と指示すればOK
- 共有方法: コピー・ダウンロード / 組織内共有 / 公開（PRO・MAXユーザのみ）

#### スキル
- Claudeが特定のタスクのパフォーマンスを上げるための専門知識パッケージ（指示・スクリプト・リソースファイル）
- ファイル作成機能（Excel・PDFなど）もスキルによって実現されている
- 無料プランでは使用不可（PRO・MAXユーザのみ）

**2種類のスキル:**
| 種類 | 説明 |
|------|------|
| Anthropic Skills | Anthropic社が作成・管理するスキル（Excel・PowerPoint作成なども含む） |
| カスタムスキル | 自分や組織が作成する、ワークフロー固有のスキル（個人アカウントのみに保持） |

**カスタムスキルの作成手順:**
1. 新しいチャットで作りたいものをClaudeに伝える
2. Claudeの質問に答える（ワークフローのインタビュー）
3. 参考資料をアップロード（テンプレート・スタイルガイドなど）
4. 完成したスキルをZIPでダウンロード
5. 設定 > 機能 > スキル からアップロード

**セキュリティの注意点:**
- 信頼できるソースからのみカスタムスキルをインストールすること
- Anthropicの組み込みスキルはテスト・保守済み
- カスタムスキルは個人アカウントのみに保持される

#### プロジェクトとスキルの違い

| 項目 | プロジェクト | スキル |
|------|-------------|--------|
| 目的 | 知識を蓄える（Claudeの参考文献） | Claudeが実行するプロセスを定義する |
| 最適 | 長期的な視点、参考資料、チーム共同作業 | 反復可能なワークフロー、複数ステップのタスク |
| 例 | 顧客ハブ、リサーチパートナー | ブランドガイドライン適用、ブログ記事作成、PDF作成 |
| 持続性 | プロジェクト内の全チャットで利用可能 | スキルが発動された際に適用される |
